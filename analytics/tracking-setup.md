# Analytics & Tracking Setup - Steel Snob

## Required Accounts

### Google Search Console
- [ ] Verify domain ownership (DNS TXT record or HTML tag)
- [ ] Submit sitemap: https://steelsnob.com/sitemap_index.xml
- [ ] Check indexing status of all pages
- [ ] Set up email alerts for crawl errors

### Google Analytics 4
- [ ] Create GA4 property for steelsnob.com
- [ ] Install tracking code (via Rank Math or Site Kit plugin)
- [ ] Set up events:
  - Affiliate link clicks (outbound click to amazon.com)
  - Email signup
  - Scroll depth (25%, 50%, 75%, 100%)
  - Time on page
- [ ] Create conversions for affiliate clicks

### Rank Tracking
**Free option:** Google Search Console (delayed data, 3-month history)
**Paid options (recommended):**
- SE Ranking ($30/mo) - good value, tracks daily
- Ahrefs Lite ($99/mo) - if budget allows, best for competitor analysis
- SERPRobot (free for 5 keywords) - quick spot checks

### Key Metrics to Track Weekly

| Metric | Tool | Target Month 1 | Target Month 3 |
|--------|------|----------------|----------------|
| Organic sessions | GA4 | 500-1,000 | 10,000 |
| Indexed pages | GSC | 40+ | 100+ |
| Keywords ranked (top 100) | GSC/Ahrefs | 100+ | 500+ |
| Keywords ranked (top 10) | GSC/Ahrefs | 10+ | 50+ |
| Avg. position | GSC | 40-60 | 15-25 |
| Click-through rate | GSC | 2-3% | 4-6% |
| Affiliate clicks | GA4 | 50+ | 500+ |
| Backlinks | Ahrefs/GSC | 20+ | 80+ |
| Domain Rating | Ahrefs | Check baseline | +5-10 |

### Monthly Reporting
Create a simple dashboard or spreadsheet tracking:
1. Total organic traffic
2. Top 10 performing pages
3. New keywords ranking
4. Affiliate revenue (Amazon Associates dashboard)
5. Content published count
6. Backlinks acquired
